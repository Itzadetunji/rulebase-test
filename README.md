# Rulebase

Compliance review stack: a **Bun + Hono API** (`api/`) and a **React + Vite UI** (`ui/`). The API evaluates interaction data for UDAAP-style rules; the UI uploads CSVs and inspects results.

**Quick start**

- API: `cd api && bun install && bun run dev` → [http://localhost:3000](http://localhost:3000) - Live at [https://hemline.app](https://hemline.app)
- UI: `cd ui && bun install && bun run dev` → [http://localhost:5173](http://localhost:5173) - Live at [https://rulebase-test.vercel.app/](https://rulebase-test.vercel.app/)

See `api/README.md` and `ui/README.md` for details.
